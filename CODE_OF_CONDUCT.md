# 贡献者行为准则

## 我们的承诺

为营造开放、友好的社区，我们承诺让每一位参与者都能免受骚扰，无论其年龄、体型、身心障碍、族裔、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、宗教或性取向。

## 我们的准则

有助于营造积极环境的行为包括：

- 使用友善、包容的语言
- 尊重不同的观点与经验
- 优雅地接受建设性批评
- 以社区利益为重
- 对其他成员保持同理心

不可接受的行为包括：

- 使用性化的语言或图像，以及任何形式的性骚扰
- 挑衅、侮辱或贬损性评论，人身或政治攻击
- 公开或私下的骚扰
- 未经许可发布他人的私人信息
- 其他在专业场合中合理认定为不当的行为

## 责任与执行

项目维护者有权移除、编辑或拒绝不符合本准则的评论、提交、代码、issue 及其他贡献。

如遇违规行为，请联系项目维护者（NJUPT-SAST）。所有投诉都会被审查并作出适当回应，投诉者信息将被保密。

## 归属
